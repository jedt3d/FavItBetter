# Specialized Tracks

FavItBetter may use specialized tracks when a feature needs deeper focus.

## Current Tracks

- UX Architecture Track: mental model, information architecture, workflows, interaction patterns, component inventory, and screen map.
- Data Model Track: proposed future track for bookmark entities, source snapshots, sync events, tombstones, conflicts, and plugin manifests.
- Integration Track: proposed future track for browser storage formats, mobile platform limits, and Shared Drive providers.
- Release Readiness Track: proposed future track for packaging, migration, rollback, and support.

## Rule

Track outputs must connect back to the same canonical docs, feature specs, decisions, tests, and release notes.
